flomi
=====
Spiel-Projekt für die Verwaltung von Flohmärkten. Es geht primär um das Austesten von verschiedenen 
Technologien:

* bndtools
* Schlankes OSGi (ohne Blueprint)
* JPA mit Transactions aber ohne Spring
* AngularJS UI
